The following two folders contain the graphical version of snake made using SFML and C++.
One is an executable file generated for 64-bit PC using Visual Studio 17 and runs without console.
The other contains the code for the above program.
To execute the code you would have to download sfml library files which is available for free at  https://www.sfml-dev.org/download.php
and link them in your compiler.
The dlls then need to be pasted in the folder in which the exe is present.
All the files in this have been created from scratch.
